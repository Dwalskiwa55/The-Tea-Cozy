HTML CHEATSHEET

Tables

Tag Name Use
<table> Table Use to create a table in html
<thead> Table Head Set around the head of your table
<tbody> Table Body Set around the body of your table
<tr> Table Row Set around the elements of each row
<td> Table Data Each cell container for your data
<th> Table Header Like td, but is bold and centered. Usually used for the table head
<tfoot> Table Foot Set around the foot of your table (sum up, add, etc.)
colspan Column Span used within the <td> and the <th> to extend to more columns
rowspan Row Span Same as Colespan but for rows instead
